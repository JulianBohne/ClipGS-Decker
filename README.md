# ClipGS Decker

## Getting Started
0. Make sure this repo was cloned with git lfs enabled
1. Download and install [decker](https://decker.cs.tu-dortmund.de/)
  - I am using `branch: develop, commit: 2062dfedf2dd416ed92313f27dbf68209f606bdd`
2. Make sure your browser supports WebGPU (and has WebGPU enabled)
  - You can use [compute.toys](https://compute.toys/) to verify / check this
3. Run `decker --server` in the root directory of this repository
4. Open `localhost:8888/ClipGS-deck.html` in your browser
  - Note Demo on Slide "Gaussian Parameters" does not seem to work in all browsers / environments
